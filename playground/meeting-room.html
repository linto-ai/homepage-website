Navigation à la voix

Comment ça marche ?

Bienvenue sur la page de démonstration de LinTO dans un navigateur web. La navigation et les interactions sur cette page se font à la voix, grâce à l'assistant LinTO. Pour povoir utiliser les commandes vocales, veuillez-suivre les indications suivantes.

1/ Autoriser l'utilisation du microphone sur la page web

Lorsqe vous chargez la page web, un message de votre navigateur devrait apparaitre, vous demandant d'autoriser l'utilisation de votre micropohone sur cette page web.

Veuillez autoriser l'utilisation du microphone.

2/ Prononcer le mot-réveil pour passer en mode "écoute"

Une fois l'uilisation de votre microphone autorisé, vous devriez voir LinTO apparaitre dans le coin suppérieur gauche de la page. lorsqu'il est en état de "repos", cela signifie qu'il attend qu'on prononce le mot de réveil "LinTO" avant d'écouter les commandes proncées.

Dites : "LinTO"

Lorsque le mot de réveil est capté par "LinTO", vous devriez entendre un bip sonore et voir LinTO passer en état d'écoute

3/ Prononcer une commande

Lorsque le mot de réveil a été capté et que LinTO est en mode écoute, prononcez une commande parmis celle renseignées sur la page.

Dites : "bloc suivant"
